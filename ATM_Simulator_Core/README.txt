ATM Simulator - Core Java (Eclipse project)
==========================================

Project: ATM_Simulator_Core
Location inside ZIP: ATM_Simulator_Core/

How to import into Eclipse:
1. Download and unzip the file.
2. In Eclipse: File -> Import... -> Existing Projects into Workspace -> Select root directory -> ATM_Simulator_Core
3. Click Finish. Eclipse will detect the project.
4. Right-click src/atm/ATMMain.java -> Run As -> Java Application
